# emp
A file packager.

## Future
I look in the future to add functionality to compress (in .gz format).

## Installing

### Prerequisites
- [Git](https://git-scm.com/)

### Steaps:
```
  $ git clone https://github.com/lucasdc6/emp.git
  $ make
  $ make install
```
You may need to give sudo permissions.
## Uninstalling

### Steaps
```
  $ make uninstall
```
You may need to give sudo permissions.
