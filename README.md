# emp
A file packager.

## Future
I look in the future to add functionality to compress (in .gz format).

## Installing

### Prerequisites
- [Git](https://git-scm.com/)

### Steaps:
```
1- $ git clone 
2- $ make
3- $ make install
```
You may need to give sudo permissions.
## Uninstalling

### Steaps
```
1- $ make uninstall
```
You may need to give sudo permissions.
